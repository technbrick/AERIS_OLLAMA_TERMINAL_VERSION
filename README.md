To get started(manual way):

1. Download Ollama. If you don't have ollama installed, click this link to download ollama:  https://ollama.com/download
2. Install the required packages. Go to requirements.txt, copy all the libraries, create a new file named install.txt, paste the libraries and run this command in your command prompt: pip install -r install.txt
3. Copy the code in agent
4. Pull the ollama models. Once you have ollama installed, you should be able to type in this command: ollama pull qwen2.5:1.5b and ollama pull qwen3:1.7b
    The models are only 1 giga each. Should run on cpu quite fast.

   
5. Voila, run the model



Quick Start (Git Clone Method)

Instead of manual setup, you can install AERIS in seconds using Git:

Clone the repository:

git clone https://github.com/technbrick/AERIS_OLLAMA_TERMINAL_VERSION.git
cd AERIS_OLLAMA_TERMINAL_VERSION

Install dependencies:
pip install -r requirements.txt

Install and run Ollama:
Download from: https://ollama.com/download

Then pull the required models:

ollama pull qwen2.5:1.5b
ollama pull qwen3:1.7b

Run AERIS:
python main.py
